Добро пожаловать в CRUD приложение для работы с базой данных ресторана!

Приложение предоставляет следующие функции:

  • Создание, чтение, обновление и удаление (CRUD) записей в базе данных
  • Фильтрация записей по различным полям
  • Сортировка столбцов в таблице
  • Регистрация на странице для возможности зарегистрироваться, войти и выйти
  • Гистограмма для визуального отображения количества записей в каждой таблице
  • Поле об авторе для связи со мной
